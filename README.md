# 💍 Wedding Venues Map — Ireland & UK

Interactive map of wedding venues across Ireland and the UK.

**[View the map →](https://santaclawone.github.io/wedding-venues/)**

## Data

- **351 geocoded venues** (44 Ireland, 307 UK)
- Sources: weddingdates.ie, weddingplanner.co.uk
- Fields: venue name, county, location, website, email (where available), GPS coordinates
- Filter by country with one click

## Tech

Leaflet.js + MarkerCluster — single HTML page, data loads as JSON.
